## 网络设置

### 查看所有的网卡

```sh
# 主要寻找网卡的 【默认网关】，如果没有，则使用【DHCP 服务器】两者大概率一致
ipconfig /all
```

### 查看路由设置

```sh
# 主要用于查看网卡的序号
route print
```

### 清空原来的设置

```sh
# 清空默认方案
route delete 0.0.0.0
```

### 添加默认路由方案

```sh
route add -p 0.0.0.0 mask 0.0.0.0 10.169.143.115 metric 1
# -p：永久
# 第一个 0.0.0.0： 目标ip，0.0.0.0代表所有
# 第二个 0.0.0.0：子网掩码，0.0.0.0 代表所有
# 10.169.143.115：要走网卡的网关
# metric 1 优先级，越小优先级越高,但是会先走单一路由方案，如果找不到才会找默认路由方案
```

### 添加 单一ip路由

```sh
route add -p 172.20.238.221 mask 255.255.255.255 192.168.1.254 metric 5 if 8
# 172.21.183.196：目标ip
# 255.255.255.255：子网掩码，此标识只针对这一个目标ip
# 192.168.1.254：要走网卡的网关
# metric 5：优先级
# if 8 要的网卡序号，【route print】命令中可以查看
```

```sh
route add -p 172.20.238.221 mask 255.255.255.255 192.168.1.254 metric 5 if 8
route delete 172.20.238.221
```
